# NBPlayer
NBPlayer：一个能播放播放无B帧视频流的播放器

#### 说明：此NB非彼NB，而是NO B Frame。即本播放器用于播放不带有B帧的视频流。


#### 用途：播放不带B帧的视频，那么最能适配的需求场景莫过于直播流，尤其是直播拉流。
